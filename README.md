# Fullstack Demo - Chat App

### API
* **:node** = message, user
* **:_id** = mongodb id

Available routes
==

| Method | route           |                 |
| ------ | --------------- | --------------- |
| GET    | /:node          | list collection |
| GET    | /:node/:_id     | read            |
| POST   | /:node          | create          |
| PUT    | /:node/:_id     | update          |
| DEL    | /:node/:_id     | delete          |
| POST   | /create_account |                 |
| POST   | /login          |                 |
| POST   | /logout         |                 |
